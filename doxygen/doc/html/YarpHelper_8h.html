PolyDriverDescriptor BipedalLocomotion::RobotInterface::constructGenericSensorClient ( std::weak_ptr< const BipedalLocomotion::ParametersHandler::IParametersHandler >  handler)

Helper function that can be used to build a GenericSensorClient device.

Parameters
handlerpointer to a parameter handler interface.
Note
the following parameters are required by the function
Parameter Name Type Description Mandatory
description string Description of the genericSensorClient. It is the device name Yes
remote_port_name string Name of the port associate to the remote Yes
local_prefix string Prefix of the local port (e.g. the application name) Yes
local_port_name_suffix string Suffix of the local port. The local port name is /<local_prefix><local_port_name_suffix> Yes
The genericSensorClient device is implement in whole-body-estimators.
Returns
A PolyDriverDescriptor. If one of the parameters is missing an invalid PolyDriverDescriptor is returned.
