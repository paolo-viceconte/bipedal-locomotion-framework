Detailed Description

TaskSpaceInverseDynamics implements the interface for the task space inverse dynamics.

Please inherit this class if you want to implement your custom Task TSID. The TSIDState is a struct containing the joint acceleration, joint torques and contact wrenches. The TaskSpaceInverseDynamics can be used to generate the desired joint torques to be sent to the low-level torque controllers. Here you can find an example of the TaskSpaceInverseDynamics interface.
